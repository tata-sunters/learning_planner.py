# learning_planner.py
# Adaptive Learning & Dynamic Time Planner for EGE Mathematics

Программная реализация адаптивного модуля динамического планирования учебного времени для персонализированной подготовки к ЕГЭ по профильной математике. В основе системы лежит концепция непрерывного измерительного контура знаний на базе современной теории тестирования (IRT) и динамического перераспределения временных ресурсов в условиях жестких дегенеративных факторов (пропуски занятий, сжатые сроки).

## 🚀 Основные возможности

*   **Дидактический граф зависимостей:** Содержание КИМ ЕГЭ декомпозировано на разделы (Алгебра, Геометрия, Теория вероятностей). Доступ к сложным темам блокируется экспертными порогами открытия до усвоения базовых компетенций.
*   **Математическое ядро IRT (Item Response Theory):** Оценка способностей пользователя ($\theta$) и сложности задач ($\beta$) в единой шкале логитов с использованием одномерной логистической модели Раша[cite: 1].
*   **Двухуровневый контур адаптации:**
    1. *Макро-уровень (Калибровка):* Первичный расчет параметров методом совместного максимального правдоподобия (JMLE) с численной оптимизацией Ньютона–Рафсона[cite: 1].
    2. *Микро-уровень (Трекинг):* Пошаговое обновление рейтинга знаний по результатам домашних заданий на базе модифицированного алгоритма Elo[cite: 1].
*   **Динамический тайм-менеджмент с ROI-фильтром:** Автоматический расчет месячных фондов времени и разбивка на 10-дневные интервальные такты[cite: 1]. При падении сроков до критических система переходит в режим «дефицита времени» ($shortage = True$), принудительно отсекая нерентабельные сложные задачи в пользу закрепления гарантированных баллов[cite: 1].
*   **Интерактивный GUI (Single Window):** Графический интерфейс на `matplotlib` с постраничной навигацией и визуализацией потактовых планов и графиков перераспределения нагрузки[cite: 1].

## 🛠️ Технологический стек

*   **Язык разработки:** Python 3[cite: 1]
*   **Библиотеки:** `matplotlib` (визуализация), `math`, `random`, `datetime` (стандартная библиотека)[cite: 1]

## 🏗️ Архитектура проекта и надежность

Программный комплекс спроектирован с учетом высоких требований к отказоустойчивости и академической точности:
*   Реализована строгая циклическая валидация пользовательского ввода с перехватом исключений `ValueError`[cite: 1].
*   Внедрена защита математических вычислений от переполнения памяти `OverflowError` при расчете экспоненциальных функций модели Раша[cite: 1].
*   Предусмотрена программная коррекция экстремальных результатов тестирования (0% и 100% успеваемость) для предотвращения ухода алгоритма JMLE в бесконечный цикл[cite: 1].
*   Архитектура обладает высокой резистентностью к пропускам дедлайнов — система автоматически перестраивает траекторию без сбоев в расчете расписания[cite: 1].

1. Склонируйте репозиторий:
```bash

   cd adaptive-math-planner

Bash
pip install -r requirements.txt
(Файл requirements.txt должен содержать библиотеку matplotlib)[cite: 1].

Bash
python adaptive_planner.py
